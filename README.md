# My LaTeX CV

Welcome to my LaTeX CV repository. This repository contains the source code for my CV, which is written in LaTeX.

## Usage

To compile the CV, you can use the following command:

```sh
pdflatex cv.tex
biber cv
pdflatex cv.tex
```
